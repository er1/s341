FIG# shows the system sequence diagram for UC6 (Browse Course List). The user activates the "Browse Course List" feature causing the the application to query the database for all the courses and return the list to the user. The user may filter the list as often as desired causing a lookup to the database for a filtered course list which again is returned to the user. Finally, the user selects a course which causes the system to query the database for the the details on the course and returns the result to the user.
